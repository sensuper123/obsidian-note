# 创建节点
```javascript
document.createElement('div')
document.createComment('我是注释节点')
document.createTextNode('我是文本节点')
```
# appendChild
```javascript
//appendChild 在Node.prototype上 第一个参数是节点
var li = document.getElementTagName('li')[0]
var text = document.createText('我是文本节点')
li.appendChild(text)
//appendChild除了有增加子节点的功能，还有剪切功能

```
# insertBefore
```javascript
a.insertBefore(b,c)
a --- 父节点
b --- 要插入的节点
c --- 插入到c节点之前
```
# removeChild 、remove
```javascript
//removeChild 删除节点
a.removeChild(p)
//将a的子节点p删除，此删除操作只是把p从dom树上删除，返回了被删除的节点
---------------------------------------
p.reomve()
//自杀，把dom树上的p节点删除，返回undefined
```
