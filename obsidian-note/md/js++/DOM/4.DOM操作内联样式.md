# element.style
```javascript
1. Dom不直接操作css样式表，而是操作元素style属性
2. 保留字要加css，如float,在强类型语言float是保留字,设置：cssFloat
element.style.width = '200px'
element.style.position = 'absolute'
element.style.cssFloat = 'left'
```
# window.getComputedStyle
```javascript
//封装获取元素样式函数
//element.offsetWidth 会把padding加进去，如果要持续改变width的值，padding会对width进行干扰
getComuptedStyle第二个参数是填伪元素的，只读
 function getTypes(ele, prop) {
      if (window.getComputedStyle) {
        if (prop) {
          return window.getComputedStyle(ele, null)[prop]
        } else {
          return window.getComputedStyle(ele, null)
        }
      } else {
        if (prop) {
          return ele.currentStyle[prop]
        } else {
          return ele.currentStyle
        }
      }
    }
```
# 当大量操作样式
```javascript
//当大量操作样式时，最好的是添加类，然后把样式放到css上写，因为element.style. 是会耗费性能的，代码也不够优雅
odiv.style.width = '100px'
odiv.style.height = '100px'
odiv.style.backgroundColor = 'red'
active {
	width:100px
	height:100px
	background-color:'red'
}
odiv.className += ' active'
```
